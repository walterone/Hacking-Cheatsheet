## Curl, Wget, Axel
```bash
#Con la '-O' salva il file con il nome originale senza specificare destinazione
curl http://ip:port -O
#Con la '-o' salva il file alla destinazione specificata
curl http://ip:port -o <path-dest>

#In wget invece la '-O' specifica il path di destinazione
wget -O http://ip:port <path-dest>
```
